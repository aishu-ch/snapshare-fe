# **SnapShare**

An app where you can 
- post pictures with caption
- edit and delete posts
- like and unlike posts
- comment and delete comment on posts
- search for people
- follow and unfollow people

## **Technologies used**

### **Frontend**

- React
- Typescript
- Axios
- TailwindCSS
